# Bash Scripting Suite

This project is part of the Capstone for Linux System Programming given by wipro.  
It provides a set of Bash scripts to automate common Linux administration tasks.

## The Features are given Below:

1. **Backup Script** – Creates timestamped backups of system or user directories.
2. **System Update Script** – Automatically updates and cleans packages.
3. **Cleanup Script** – Deletes old log files.
4. **Log Monitoring Script** – Scans logs for critical errors or failures.
5. **Interactive Menu** – A single interface to access all scripts easily.
